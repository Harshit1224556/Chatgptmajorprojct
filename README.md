📌 About The Project

This project is a real-time AI-powered chat application inspired by ChatGPT.

It allows users to:

💬 Start multiple conversations

🔁 Switch between old chats

🧠 Get AI-powered responses

🔊 Listen to AI responses (Voice Feature)

🔐 Login & maintain session

⚡ Experience real-time communication using WebSockets

Even if you return after months, your chat history is still available.

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Harshit1224556/Chatgptmajorprojct.git
cd Chatgptmajorprojct
2️⃣ Backend Setup
cd backend
npm install

Create a .env file inside backend:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_key

Run backend:

npm start
3️⃣ Frontend Setup
cd frontend
npm install
npm run dev
